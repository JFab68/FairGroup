# FAIR Group Website\n\n## Quick Start\n\n### Prerequisites\n- Node.js 18+\n- PostgreSQL 14+\n\n### Setup\n1. Save this script as `generate-project.sh`\n2. `chmod +x generate-project.sh && ./generate-project.sh`\n3. `cd backend && npm install && npm run migrate`\n4. `cd frontend && npm install`\n5. Create admin user via SQL\n6. Run both servers: `npm run dev` in separate terminals\n\n## Features\nPublic site, member auth, 6 subcommittees, chair powers, attendance tracking, CSV tools\n\n## Tech Stack\nBackend: Node.js, Express, PostgreSQL | Frontend: React, React Query
